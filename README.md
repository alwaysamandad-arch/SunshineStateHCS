# Sunshine State Home & Commercial Services

Professional home and commercial services website with integrated booking form.

## Website Features

- **Responsive Design**: Mobile-first approach works on all devices
- **Service Booking Form**: Integrated with Netlify Forms for automatic email notifications
- **Professional Layout**: Clean, modern design with clear service information
- **Emergency Contact**: 24/7 phone availability highlighted throughout
- **Service Pages**: Detailed information about each service offering

## Services Offered

- 🔧 General Repairs
- 🚿 Plumbing
- ⚡ Electrical
- 🎨 Painting
- 💧 Water Damage Restoration

## Form Handling

The booking form is configured with:
- **Netlify Forms** for automatic submission handling
- **Email Notifications** sent to LevlarFlex@gmail.com
- **Honeypot Field** for spam prevention
- **Date/Time Selection** with radio buttons
- **Auto-confirmation** pages

## Deployment

1. Push to GitHub
2. Connect repository to Netlify
3. Netlify will automatically build and deploy on every push
4. Forms are automatically enabled and monitored in Netlify dashboard
